一些爬虫 -- some crawlers

2020-02-16 • 分类: python • 标签: crawler, learning, python

my crawlers

因为新型冠状病毒肆虐的原因，寒假之后开始就一直宅在家里。这段时间里，我写了几个爬虫https://github.com/Hexa4C/crawlers，爬了一些资源。

requirements

爬虫使用的python版本和包主要是这些：

  • python >= 3.7
  • requests >= 2.22.0
  • beautifulsoup4 >= 4.8.2

crawlers

本库内包含这些爬虫

  • 腾讯较真平台COVID-19相关传言的辟谣与证实内容爬虫

    这个爬虫是用来爬取疫情期间腾讯较真平台上面的一些关于传言的澄清和证实的。

    结果以json的形式存储在article.json文件内。每个条目包含以下内容：

    • title：传言标题
    • subtitle：传言副标题
    • label：标签
    • point：描述
    • source：来源

  • 人民教育出版社中小学生各类教材的电子版PDF批量下载爬虫

    由于疫情，人民教育出版社推出了中小学生各类教材的电子版PDF免费下载，为了帮别人批量下载一些教材，写了这个爬虫。下载下来的书会按文件夹存储在store_path下。由于书数量不少，如果真有人想用这个脚本下载最好改改。因为（时间原因）我懒得改了，所以没有加上可以按关键字来下载之类的功能。
